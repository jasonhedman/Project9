Game Show App

Skills: HTML, CSS, JavaScript, JavaScript Events, Template Literals, JavaScript Loops

Given a set of base styles, I created a browser version of the game "Wheel of Success" , a word guessing game where players try to guess a randomly generated phrase. The phrase is randomly selected from a customizable array of phrases, then split into an array of letters, and put onto the game board. The user attempts to guess the phrase until they complete the phrase, or use all of their 5 lives. When either of these instances occur, the user is brought to a screen where they can restart with a new phrase.
